
# Programación Backend

Este es un proyecto para la entrega del desafío "Mocking y manejo de errores" en el curso de Programación Backend


**Para iniciar en "Modo Development"**

``` npm run dev ```
<br>
``` npm run dev -- -m dev ```

**Para iniciar en "Modo Production"**

``` npm run dev -- -m prod ```
<br>
``` npm start -- -m prod ```
